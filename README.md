# Praktisk Laboration (Moment 1.2 - TypeScript)

## Beskrivning
En enkel webbapplikation för att hantera kurser. 
Användaren kan:
- Visa kurslistan
- Lägga till nya kurser
- Sökfunktion
- Sortera på kurskod, namn, progression eller syllabus

Kurser sparas i `localStorage`.

## Tekniker
- TypeScript
- Vite
- DOM-manipulation
- localStorage
